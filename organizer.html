        Study Process Reference
      

      
This tool is designed to eliminate AI hallucinations by extracting and fixing PDF formulas deterministically before feeding them to an LLM.

      
      
1. Select a Chapter

      
Books are automatically chunked into logical chapters. Select one from your Library.


      
2. Choose a Study Mode

      
    
        
  • Deep Dive: Step-by-step breakdown of dense math.
    • 
        
  • Exam Prep: Generates practice questions based on formulas.
    • 
        
  • Quick Review: Cheat-sheet style summary.
    • 
        
  • Socratic: Tests you with leading questions.
    • 
      


      
3. Interact with the LLM

      
Click Generate Study Prompt. Copy the resulting prompt into ChatGPT, Claude, or Gemini. Ask follow-up questions if parts remain unclear!
